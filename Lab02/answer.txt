Question: If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?

Trả lời:
Không, vì title và category đều cùng kiểu dữ liệu là String, mà JAVA chỉ biết phân biệt constructor theo số lượng và thứ tự kiểu dữ liệu của các biến đầu vào.
